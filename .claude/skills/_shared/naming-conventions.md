# Naming Conventions

## Namespace: `Company.PackageName`
Хорошо: `SashaRX.PrefabDoctor`. Плохо: `PrefabDoctor`.

## package.json: `com.company.package-name` (kebab-case)

## asmdef: `Company.PackageName.Editor`

## Файлы: PascalCase, один класс = один файл
EditorWindow → `*Window.cs`, PropertyDrawer → `*Drawer.cs`, Tests → `*Tests.cs`

## MenuItem: `Tools/Package Name/Action`
